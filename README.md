This Project is build on the following technologies
1.Front End: HTML, CSS, Bootstrap, JSP 
2.Back End: Java J2EE(Servlet), JDBC 
3.Database: Mysql 
4.Server: Apache Tomcat 
5.Tool: Eclipse IDE, Mysql Workbench
